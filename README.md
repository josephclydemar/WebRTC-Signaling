# WebRTC Signaling Experiment
### WebRTC Signaling server experiment in TypeScript.

<br>

---

<br>
<br>

#### Start the Development Environment:
##### Step 1:
```
npm run prettier-watch
```

##### Step 2:
```
npm run ts-watch
```

##### Step 3:
```
npm run webpack
```

##### Step 4:
```
npm run dev
```
